# Plan: Pre-commit Hook (v1.1.0)

## Goal

Enable developers to use `fetchext` as a pre-commit hook to audit extensions before committing them.

## Strategy

### 1. Configuration

- Create `.pre-commit-hooks.yaml`.
- Define hooks:
  - `fext-audit`: Run `fext audit`.
  - `fext-scan`: Run `fext scan`.
  - `fext-analyze`: Run `fext analyze --complexity`.

### 2. Documentation

- Add "Pre-commit Hook" section to `README.md` and `docs/integration.md` (or `docs/cli.md`).

## Steps

1. Create `.pre-commit-hooks.yaml`.
2. Update docs.
