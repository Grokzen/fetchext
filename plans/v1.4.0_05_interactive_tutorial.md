# Plan: Interactive Tutorial (v1.4.0 Item 5)

## Goal
Implement `fext tutorial` command to launch an interactive TUI walkthrough that teaches users how to use the tool.

## Requirements
1.  **Command**: `fext tutorial`
2.  **Interface**: TUI using `textual` (since we already use it for `fext ui`).
3.  **Content**:
    *   Introduction to `fext`.
    *   Step-by-step guide for common commands:
        *   Downloading extensions.
        *   Searching.
        *   Inspecting/Analyzing.
        *   Configuration.
    *   Interactive elements (e.g., "Press Next", "Try it").
4.  **Style**: Friendly, educational, easy to navigate.

## Implementation Steps
1.  **Create Module**: `src/fetchext/tutorial.py`.
    *   Define a `TutorialApp` class inheriting from `textual.app.App`.
    *   Create screens for each tutorial step.
    *   Use `Markdown` widget for content.
2.  **Update CLI**: `src/fetchext/cli.py`.
    *   Add `tutorial` subcommand.
    *   Call `run_tutorial()` from `fetchext.tutorial`.
3.  **Content Creation**:
    *   Draft the tutorial text (Markdown format).
4.  **Tests**:
    *   Unit tests for the tutorial app structure (using `textual` testing tools if possible, or basic instantiation tests).

## Verification
1.  Run `fext tutorial` and navigate through the steps.
2.  Ensure it exits cleanly.
