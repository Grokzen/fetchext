# Plan: Interactive TUI (v0.9.0)

## Goal

Implement a terminal user interface for browsing and downloading extensions.

## Design

- **Module**: `src/fetchext/tui.py`
- **Library**: `textual` (needs to be added to dependencies).
- **Features**:
  - Search bar.
  - List of results.
  - Details view.
  - Download button.
- **Integration**:
  - Add `fext ui` command.

## Implementation Steps

1. Add `textual` to `pyproject.toml`.
2. Create `src/fetchext/tui.py`.
   - Define `ExtensionApp` class inheriting from `textual.app.App`.
   - Layout: Header, Search Input, Data Table (results), Footer.
3. Update `src/fetchext/cli.py` to add `ui` command.
4. Add tests (mocked).

## Dependencies

- `textual`
