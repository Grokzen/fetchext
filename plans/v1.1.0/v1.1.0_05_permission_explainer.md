# Plan: Permission Explainer (v1.1.0)

## Goal

Provide a tool to explain what specific browser extension permissions allow, helping users understand the security implications.

## Strategy

### 1. Data Source

- Create a dictionary of permissions and their descriptions.
- Source: MDN Web Docs, Chrome Developer Docs.
- Store in `src/fetchext/analysis/permissions_db.py`.

### 2. New Command

- `fext explain <permission>`
- Example: `fext explain tabs` -> "Allows the extension to access your tabs, including URL and title."

### 3. Implementation

- Create `src/fetchext/analysis/explainer.py`.
- Implement fuzzy search for permissions.
- Output formatted explanation with risk level (if available).

### 4. Integration

- Update `cli.py`.
- Update docs.

## Steps

1. Create `src/fetchext/analysis/permissions_db.py`.
2. Create `src/fetchext/analysis/explainer.py`.
3. Update `cli.py`.
4. Add tests.
