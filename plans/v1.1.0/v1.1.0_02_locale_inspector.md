# Plan: Locale Inspector (v1.1.0)

## Goal

Implement a tool to inspect the internationalization (i18n) support of an extension. This involves listing supported locales and extracting messages from `_locales/<locale>/messages.json`.

## Strategy

### 1. New Command

- `fext locales <file> [--json]`
- Or `fext inspect --locales <file>`?
- The roadmap says `fext locales <file>`.
- I'll implement `fext locales` as a top-level command (or maybe under `inspect`? No, `inspect` is for manifest).
- Let's stick to `fext locales`.

### 2. Implementation

- Create `src/fetchext/analysis/locales.py`.
- Use `PartialFileReader` / `zipfile` to list directories starting with `_locales/`.
- Parse `messages.json` for each locale.
- Output:
  - List of locale codes (e.g., `en`, `fr`, `es`).
  - Count of messages per locale.
  - (Optional) "Default Locale" from manifest.

### 3. Integration

- Update `cli.py`.
- Update docs.

## Steps

1. Create `src/fetchext/analysis/locales.py`.
2. Update `cli.py`.
3. Add tests.
