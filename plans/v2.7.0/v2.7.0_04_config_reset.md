# Plan: Config Reset Command

## 1. Introduction
Users may misconfigure `config.toml` leading to runtime errors. A quick way to reset to factory defaults is needed.

## 2. Objectives
- Implement `fext config reset`.
- Backup the existing config before overwriting.
- Restore default configuration values.

## 3. Implementation Details

### 3.1. CLI Command
- Add `reset` subcommand to `config` group in `src/fetchext/commands/config.py`.

### 3.2. Logic
1.  Locate `config.toml` (usually `~/.config/fext/config.toml`).
2.  If it exists, rename it to `config.toml.bak.<timestamp>`.
3.  Write the default configuration (defined in `src/fetchext/config.py` or a template) to `config.toml`.
4.  Print success message with path to backup.

### 3.3. Safety
- Ask for confirmation (`Are you sure? [y/N]`) unless `--force` is used.

## 4. Verification
- **Test**: Corrupt the local config file. Run `fext config reset`.
- **Assert**: Config file is valid again. Backup exists.
