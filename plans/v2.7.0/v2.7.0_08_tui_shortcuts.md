# Plan: TUI Keyboard Shortcuts

## 1. Introduction
Power users prefer keyboard navigation over tabbing or mouse clicks.

## 2. Objectives
- Add configurable keyboard shortcuts to `fext ui`.
- Support common actions: Download, Search, Quit, Switch Tabs.

## 3. Implementation Details

### 3.1. Textual Bindings
- In the `TuiApp` class, define `BINDINGS`.
- Map keys to actions:
    - `ctrl+d`: Download selected.
    - `/`: Focus search bar.
    - `q`: Quit.
    - `tab`: Cycle focus (default).
    - `j` / `k`: Navigate lists (Vim style, optional).

### 3.2. Configurability
- Allow overriding bindings in `config.toml` under `[tui.keybindings]`.
- Load these at runtime and update the app's bindings.

## 4. Verification
- **Test**: Run `fext ui`. Press `/`.
- **Assert**: Search input is focused.
