# Plan: Path Standardization (v1.5.0 Item 4)

## Goal

Ensure `pathlib.Path` is used exclusively throughout the codebase, replacing any legacy `os.path` usage. This improves code readability and cross-platform compatibility.

## Current State

- Most of the codebase already uses `pathlib`.
- There might be some lingering `os.path.join`, `os.path.exists`, `os.makedirs`, etc.

## Strategy

1. **Audit**: Search for `os.path` usage.
2. **Refactor**: Replace identified usages with `pathlib.Path` equivalents.
3. **Verify**: Run tests to ensure no regressions.

## Search Patterns

- `os.path.`
- `os.makedirs` (replace with `Path.mkdir(parents=True)`)
- `os.remove` (replace with `Path.unlink()`)
- `os.rename` (replace with `Path.rename()`)
- `os.walk` (replace with `Path.rglob` or keep if performance critical, but prefer `pathlib`)
- `open(` (replace with `Path.open()`)
