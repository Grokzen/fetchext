# Plan: User-Agent Rotation

## Goal

Implement random User-Agent rotation for network requests to avoid blocking by Web Stores.

## Specification

- **Mechanism**: Select a random User-Agent string from a predefined list for each request (or session).
- **Implementation**:
  - Create a list of modern User-Agent strings (Chrome, Firefox, Edge, Safari) in `src/fetchext/network.py`.
  - Update `get_session()` in `src/fetchext/network.py` to set a random `User-Agent` header if one isn't already set.
  - Ensure that the `User-Agent` can still be overridden if needed (though not currently exposed via CLI).

## Files to Modify

- `src/fetchext/network.py`: Add UA list and rotation logic.

## Verification

- **Unit Tests**: Verify that `get_session()` returns a session with a User-Agent header from the list.
- **Integration Tests**: Verify that requests made using the session include the User-Agent header.
